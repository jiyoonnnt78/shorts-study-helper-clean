# PDF로 저장/공유 버튼 추가

## 방식
브라우저 기본 인쇄(window.print) 활용. 서버는 전혀 관여 안 함 -> 메모리 영향 0.
사용자가 "PDF로 저장" 누르면 브라우저 인쇄 대화상자가 뜨고,
대상을 "PDF로 저장"으로 고르면 다운로드됨. 모든 브라우저/폰에서 동작.

## 메모리 관련 확인 사항 (질문 답변)
- 분석 끝난 결과 페이지에 오래 머물러도 서버 메모리 영향 없음.
  (이미 DB에 저장된 결과를 보여줄 뿐, 추가 연산/메모리 사용 없음)
- 새로고침해도 안전: page.tsx가 getVideo(id)로 DB 재조회만 함 (재분석 없음).
  analyzing 페이지도 getStatus(id)로 진행 상태만 조회 (재시작 없음).
- 즉 "새로고침 -> 재분석 -> 메모리 폭증"은 이미 일어나지 않는 구조.

## 변경 파일
- src/app/videos/[id]/page.tsx : "PDF로 저장 / 공유" 버튼 추가
- src/app/globals.css          : @media print 스타일 추가
  - .no-print 클래스: 버튼 등 인쇄 시 숨김
  - 카드 그림자 제거, break-inside:avoid로 카드가 페이지 중간에 안 잘리게

## 사용법
결과 페이지 하단 "🖨️ PDF로 저장 / 공유" 클릭
-> 브라우저 인쇄 대화상자 -> 대상을 "PDF로 저장"으로 선택 -> 다운로드
-> 그 PDF 파일을 친구에게 카톡/에어드랍 등으로 공유

## 적용
두 파일만 덮어쓰면 됨. 백엔드 변경 없음. 새 패키지 설치 없음(라이브러리 의존 없음).
