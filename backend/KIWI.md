# 키워드 추출과 Kiwi 형태소 분석기

이 서비스는 어떤 유튜브 쇼츠가 들어와도 **주제 사전 없이** 핵심 키워드를 골라낸다.
주제별 단어 목록(뷰티/게임/요리...)을 손으로 채우는 방식은 쓰지 않는다.

## 핵심 선별 방식

키워드는 단순 빈도가 아니라 **여러 신호의 가중 합**으로 고른다.
같은 단어라도 어디에 나왔느냐로 중요도가 다르기 때문이다.

신호와 가중치 (여러 신호를 균형 있게 조합, 화면 글자에 약간 더 무게):

- 화면 글자(OCR) 등장 — 자막/타이틀은 제작자가 강조한 핵심 (가장 큼)
- OCR과 말(STT) 양쪽에 등장 — 확실한 핵심
- 여러 장면(segment)에 걸쳐 등장 — 영상 전체를 관통
- 첫 장면(도입부) 등장 — 주제 선언일 확률
- 고유명사/영문 (게임명·제품명 등)
- 말(STT) 등장
- 빈도 (약한 신호)
- 의도 표현(설명/소개/축하...) 근처

이 방식 덕분에 화면에 한 번 뜬 핵심어가, 말로 여러 번 반복된 잡담을 이긴다.

## Kiwi (기본 ON)

`ENABLE_KIWI=true`(기본값)이면 한국어 형태소 분석기 Kiwi로 **명사만** 추출한다.
품사 태깅이라 조사("나를"), 부사("벌써", "정말"), 대명사, 감탄사가
불용어 사전 없이 자동으로 제외된다.

- 설치: `pip install kiwipiepy`
- 끄기: `.env`에서 `ENABLE_KIWI=false` (규칙 기반으로 동작)
- 안전장치: kiwipiepy 미설치/로드/분석 실패 시 자동으로 규칙 기반 추출로 fallback.
  Kiwi 문제로 분석이 실패(failed)하는 일은 없다.

## topic / purpose 생성

- topic은 키워드를 그냥 이어붙이지 않는다.
  [의도 카테고리] + [핵심 명사 1~2개]를 자연스러운 문장으로 합성한다.
  예: "마인크래프트와 나무 방법을 알려주는 영상인 것 같아요."
- 확신이 낮으면 단정하지 않는다: "영상 내용을 더 확인해 봐야 할 것 같아요."

## 동작 위치

- `app/services/kiwi_extractor.py` — Kiwi로 (명사, 고유명사여부) 추출 (실패 시 None)
- `app/services/text_analysis.py` — `select_key_nouns`(segment 단위 신호 합산),
  `extract_concepts`(핵심 명사 + 의도 표현 + 행동/감정)
- `app/services/explainer.py` — topic/purpose/category/confidence 생성
